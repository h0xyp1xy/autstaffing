# Сайт аутстаффинга складского персонала

Одностраничный корпоративный сайт (лендинг) для компании по аутстаффингу складского персонала.

## Технологии

- HTML5
- CSS3 (Flexbox/Grid, CSS Variables)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- Chart.js (для графиков в калькуляторе)

## Структура проекта

```
autstaffing/
├── index.html          # Главная страница
├── css/
│   └── styles.css      # Основные стили
├── js/
│   ├── main.js         # Основная функциональность
│   ├── preloader.js    # Preloader с анимацией
│   ├── calculator.js   # Интерактивный калькулятор
│   └── animations.js   # GSAP анимации
├── assets/
│   ├── images/         # Изображения
│   ├── video/          # Видео файлы
│   └── documents/      # Документы (PDF, XLSX, DOCX)
├── robots.txt
├── sitemap.xml
└── README.md
```

## Особенности

### Секции сайта

1. **Hero** - Главный экран с видео фоном
2. **Калькулятор** - Интерактивный расчет издержек
3. **Решение** - Схема разделения ответственности
4. **Преимущества** - 9 стратегических преимуществ
5. **Безопасность** - Протокол безопасности с документами
6. **Процесс** - Интерактивная дорожная карта (30 дней)
7. **Результаты** - Фильтруемые кейсы
8. **Услуги** - Модульная система услуг
9. **CTA** - Финальный призыв к действию
10. **Футер** - Контакты и информация

### Функциональность

- Preloader с SVG анимацией схемы склада
- Интерактивный калькулятор с автосохранением
- Плавные GSAP анимации при скролле
- Адаптивный дизайн (Mobile-first)
- Фильтрация кейсов
- Модальные окна для форм
- Умный чат (кнопка интеркома)
- Определение компании посетителя
- Автосохранение данных форм

## Установка и запуск

1. Клонируйте репозиторий или скачайте файлы
2. Откройте `index.html` в браузере или используйте локальный сервер:

```bash
# С помощью Python
python -m http.server 8000

# Или с помощью Node.js (http-server)
npx http-server
```

3. Откройте в браузере: `http://localhost:8000`

## Настройка

### Интеграция с CRM

В файле `js/main.js` в функции `sendToCRM()` необходимо добавить интеграцию с вашей CRM системой (AmoCRM, Bitrix24 и т.д.).

### Аналитика

Добавьте коды Яндекс.Метрики и Google Analytics в конце файла `index.html` (в комментариях указано место).

### Замена контента

- Замените placeholder изображения в папке `assets/images/`
- Добавьте видео для hero-секции в `assets/video/`
- Обновите контактную информацию в футере
- Добавьте реальные документы в `assets/documents/`
