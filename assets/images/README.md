# Изображения

## Требования к изображениям

- Формат: WebP (с fallback в JPG для старых браузеров)
- Оптимизация: сжатие без потери качества
- Lazy loading: для изображений ниже fold

## Необходимые изображения

- `hero-bg.jpg` / `hero-bg.webp` - Фоновая картинка для hero-секции (fallback)
- Иллюстрации для секций (по необходимости)

## Оптимизация

Используйте инструменты:
- ImageOptim
- Squoosh
- TinyPNG

Для WebP конвертации используйте cwebp или онлайн-конвертеры.
