# Документы

Здесь должны быть размещены следующие документы:

- `financial-model.xlsx` - Финансовая модель для скачивания
- `contract-sample.pdf` - Типовой договор аутстаффинга
- `insurance-policy.pdf` - Страховой полис
- `privacy-policy.pdf` - Политика конфиденциальности
- `brief-template.docx` - Бриф для запроса КП
- `presentation-slides.pdf` - Презентация для совета

Все файлы должны быть оптимизированы для быстрой загрузки.
