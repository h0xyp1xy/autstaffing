# Видео

## Hero-видео

Требования к видео для hero-секции:

- Формат: MP4 (H.264)
- Длительность: 15-20 секунд
- Размер: до 5 МБ (оптимизированное)
- Разрешение: 1920x1080 (Full HD)
- Ракурс: верхняя точка или плавный проезд
- Без звука или с минималистичным техно-битом

## Название файла

`warehouse-hero.mp4`

## Оптимизация

Используйте:
- HandBrake для сжатия
- FFmpeg для конвертации
- Минимальный битрейт при сохранении качества

Для фона можно использовать видео с opacity или затемняющий overlay.
